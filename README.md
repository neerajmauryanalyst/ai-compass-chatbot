# AI Compass Chatbot 

An AI literacy chatbot that explains:
- AI Basics
- AI Bias
- AI Risks
- AI Ethics
- AI Regulations

## Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Flask
- Chatbot: Rasa

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run Rasa
rasa train
rasa run

### 3. Run Backend
cd backend
python app.py

### 4. Open Frontend
Open index.html in browser
